When you upload, GitHub automatically:

```
1. Sets up Ubuntu Linux
2. Installs NASM, GCC, Make, GRUB
3. Compiles boot.asm → boot.o
4. Compiles kernel.c → kernel.o
5. Compiles all 8 drivers → .o files
6. Compiles all 5 modules → .o files
7. Compiles ui_engine.c → ui_engine.o
8. Links everything → dragon_os.bin
9. Creates ISO → dragon_os.iso
10. Uploads ISO for download
```
