     • Vendor/Device ID
     • Configuration access

  7. ✅ mouse.c         - Mouse Driver (PS/2)
     • 3-button support
     • X/Y tracking
     • Interrupt-driven

  8. ✅ rtc.c           - Real-Time Clock Driver
     • Time reading
     • Date reading
     • BCD conversion

  9. ✅ drivers.h       - Driver Header File
     • All function declarations
     • Type definitions
     • Driver interface
