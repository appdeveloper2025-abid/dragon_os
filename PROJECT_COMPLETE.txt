═══════════════════════════════════════════════════════════════════════════
                        🚀 HOW TO USE DRAGON OS
═══════════════════════════════════════════════════════════════════════════

STEP 1: BUILD
  cd dragon_os
  ./build.sh          (Linux)
  build.bat           (Windows)

STEP 2: TEST
  qemu-system-i386 -cdrom dragon_os.iso -m 512M

STEP 3: CREATE BOOTABLE USB
  sudo dd if=dragon_os.iso of=/dev/sdX bs=4M

STEP 4: BOOT ON REAL HARDWARE
  Insert USB → Restart → Select USB from boot menu

STEP 5: ENJOY!
  Explore the futuristic Dragon OS interface!
