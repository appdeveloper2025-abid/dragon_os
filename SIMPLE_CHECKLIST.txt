                    FOLLOW THIS CHECKLIST STEP BY STEP
═══════════════════════════════════════════════════════════════════════════

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 1: CREATE GITHUB ACCOUNT (5 minutes)                                │
└───────────────────────────────────────────────────────────────────────────┘

  [ ] Go to https://github.com
  [ ] Click "Sign up"
  [ ] Enter email address
  [ ] Create password (write it down!)
  [ ] Choose username
  [ ] Verify email
  [ ] Account created ✓

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 2: CREATE REPOSITORY (3 minutes)                                    │
└───────────────────────────────────────────────────────────────────────────┘

  [ ] Click "+" icon (top right)
  [ ] Click "New repository"
  [ ] Name: dragon_os
  [ ] Description: Dragon OS - Futuristic Operating System by ABID MEHMOOD
  [ ] Select "Public"
  [ ] Don't check any boxes
  [ ] Click "Create repository"
  [ ] Repository created ✓

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 3: UPLOAD FILES (10 minutes)                                        │
└───────────────────────────────────────────────────────────────────────────┘

  [ ] Click "uploading an existing file"
  [ ] Open C:\Operating System\dragon_os folder
  [ ] Press Ctrl+A (select all files)
  [ ] Drag all files to GitHub
  [ ] Wait for upload
  [ ] Type: Initial commit - Dragon OS v1.0
  [ ] Click "Commit changes"
  [ ] Files uploaded ✓

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 4: WAIT FOR BUILD (10 minutes)                                      │
└───────────────────────────────────────────────────────────────────────────┘

  [ ] Click "Actions" tab
  [ ] See "Build Dragon OS ISO" running
  [ ] Wait for green checkmark ✓
  [ ] Build completed ✓

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 5: DOWNLOAD ISO (2 minutes)                                         │
└───────────────────────────────────────────────────────────────────────────┘

  [ ] Click on completed workflow
  [ ] Scroll to "Artifacts"
  [ ] Click "dragon-os-iso"
  [ ] Download ZIP file
  [ ] Extract ZIP
  [ ] Get dragon_os.iso file ✓

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 6: CREATE BOOTABLE USB (5 minutes)                                  │
└───────────────────────────────────────────────────────────────────────────┘

  [ ] Download Rufus from https://rufus.ie/
  [ ] Insert USB drive (backup data first!)
  [ ] Run Rufus
  [ ] Click "SELECT" and choose dragon_os.iso
  [ ] Click "START"
  [ ] Wait for completion
  [ ] USB ready ✓

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 7: BOOT DRAGON OS (5 minutes)                                       │
└───────────────────────────────────────────────────────────────────────────┘

  [ ] Keep USB inserted
  [ ] Restart computer
  [ ] Press F12 during startup
  [ ] Select USB drive
  [ ] Press Enter
  [ ] Dragon OS boots! 🎉

                        🆘 QUICK TROUBLESHOOTING

═══════════════════════════════════════════════════════════════════════════

  Problem: Build failed (red X)
  Solution: Check that ALL files were uploaded

  Problem: No Actions tab
  Solution: Make sure repository is Public

  Problem: Can't download ISO
  Solution: Wait for green checkmark ✓

  Problem: USB won't boot
  Solution: Try F12, F2, ESC, or DEL key
