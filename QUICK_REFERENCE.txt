┌───────────────────────────────────────────────────────────────────────────┐
│ 🔧 BUILD TOOLS REQUIRED                                                   │
└───────────────────────────────────────────────────────────────────────────┘

  Ubuntu/Debian:
    sudo apt-get install nasm gcc make grub-pc-bin xorriso qemu-system-x86

  Fedora/RHEL:
    sudo dnf install nasm gcc make grub2-tools xorriso qemu-system-x86

  Arch Linux:
    sudo pacman -S nasm gcc make grub xorriso qemu

  Windows:
    Use WSL2 or install MinGW/MSYS2/Cygwin

┌───────────────────────────────────────────────────────────────────────────┐
│ 💿 CREATE BOOTABLE USB                                                    │
└───────────────────────────────────────────────────────────────────────────┘

  Linux:
    sudo dd if=dragon_os.iso of=/dev/sdX bs=4M status=progress
    sync

  Windows:
    Use Rufus or Etcher to write dragon_os.iso to USB

┌───────────────────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                                        │
└───────────────────────────────────────────────────────────────────────────┘

  Problem: "nasm: command not found"
  Solution: Install NASM (sudo apt-get install nasm)

  Problem: "cannot find -lgcc"
  Solution: Install 32-bit libs (sudo apt-get install gcc-multilib)

  Problem: "grub-mkrescue: command not found"
  Solution: Install GRUB tools (sudo apt-get install grub-pc-bin xorriso)

  Problem: Black screen after boot
  Solution: Try different VM or check VGA compatibility

  Problem: Build errors
  Solution: Ensure all tools installed, check error messages
