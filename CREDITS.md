   ```bash
   qemu-system-i386 -cdrom dragon_os.iso -m 512M
   ```

3. **Create Bootable USB**
   ```bash
   sudo dd if=dragon_os.iso of=/dev/sdX bs=4M
   ```

4. **Boot on Real Hardware**
   - Insert USB
   - Restart computer
   - Select USB from boot menu
